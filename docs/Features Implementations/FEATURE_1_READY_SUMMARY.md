```

Step 2: View all your orders
Step 3: Click "View Details" button
Result: → Taken to order details page
```

#### 2️⃣ Profile Page - Orders Tab (`/profile`)
```
Step 1: Go to Profile page
Step 2: Click "Orders" tab
Step 3: Click on any order card
Result: → Taken to order details page (with hover effect)
```

#### 3️⃣ Direct URL Navigation
```
Type: /order/{orderId} in address bar
Result: → Order details page opens
```
